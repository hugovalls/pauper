tournament_db es un export de mi base de datos desde XAMPP, mientras que el contenido de la carpeta tournament_web es el HTML y el php de la pagina.

para abrir el proyecto  debes tener la base de datos en phpmyadmin y la carpeta de tournament_web dentro de la carpeta de XAMPP/htdocs/
y una vez este ahí  poner en el buscador:

http://localhost/tournament_web/index.html



la practica refleja los datos extraidos de un torneo del juego de cartas Magic: the gathering llamado "el superpauper" celebrado en IngenioBCN

la practica se ha realizado para cumplir las competencias pedidas por la profesora las cuales eran :

- 3 graficas que representen datos de los datos escogidos
- un enlace a la uib 
- enlace al origen de los datos
- enlace que sirva como titulo del dashboard con el atributo href="#"

se ha priorizado a la hora de representar los datos la claredad de los mismos por encima de lo compacto que se puedan ver a diferencia de la imagen planteada en el enunciado , este compromiso se debe a que había muchos datos que representar lo que complicaba el hecho de hacerlos compactos.

se puede interactuar con las tres graficas , en la primera y la tercera si se pone el cursor por encima te dice de que arqeutipo se tratan los datos que estas mirando y en la segunda puedes escoger si quieres ver una columna en especifico o varias a la vez.
